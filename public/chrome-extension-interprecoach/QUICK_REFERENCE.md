## Session Controls

### Starting
1. Click "Start Session" (blue button)
2. Allow microphone when asked
3. Look for "Live" indicator with session timer

### During Session
- All 4 panels update automatically
- Add your own notes anytime
- Use chat box at bottom for queries
- Session timer shows elapsed time

### Ending
1. Click "End Session" (red button)
2. Everything saves automatically
3. Audio is immediately discarded (HIPAA compliant)

---

## Chat Input (Bottom)

The input box at the bottom is for:
- Manual queries
- Adding custom notes to transcript
- Searching (future feature)

Press **Enter** or click **Send** button to submit.

### Common Questions

**Q: What if transcription stops?**
A: It should auto-restart. If not, end and start a new session.

**Q: Can I edit the transcript?**
A: Not directly, but add corrections in your notes.

**Q: How do I see past sessions?**
A: Currently in the database only. Export feature coming soon!

**Q: What if a medical term isn't detected?**
A: Add it to your notes. We're constantly expanding the dictionary.

**Q: Is my data secure?**
A: Yes! HIPAA compliant - audio is never stored, only transcripts.

## Troubleshooting

### No Transcription?
1. Check microphone icon in browser address bar
2. Make sure you granted permission
3. Test mic in other apps
4. Reload extension from chrome://extensions/

### Medical Terms Not Showing?
- They only appear once per session
- Must match exact term in dictionary
- Add unlisted terms to your notes

### Notes Not Saving?
- Check you're in an active session
- Click save button manually
- Check internet connection

### Extension Not Working?
1. Reload the extension
2. Close and reopen browser
3. Check browser console (F12) for errors
